💡 Почему этот тип важен
Один из самых простых способов поднять Literacy до 120–140
Здесь не нужно думать о смысле — достаточно знать английские суффиксы, приставки и частые окончания слов. Многие набирают здесь 90–100% баллов.
📖Как выглядит задание

Тебе дают предложение с 1–3 пропусками букв в словах. Нужно выбрать правильный вариант из 4–6 предложенных.

The company has experienced rapid gro___ in recent years.

Варианты: growth / grown / growing / grows

Правильно: growth

❌Топ-ошибки русскоязычных
1. Не знают суффиксы
-tion / -sion / -ment / -ance / -ence / -ity / -ness / -able / -ible
2. Путают похожие слова
effect / affect, rise / raise, lie / lay, borrow / lend
3. Игнорируют часть речи
Нужен noun, а выбирают verb
4. Не смотрят на контекст
Слово должно логически подходить по смыслу
🎯 Стратегия на максимум
5 шагов, чтобы почти никогда не ошибаться
⚡Метод Teacher Luke + адаптация
  1. Сначала прочитай всё предложение целиком — не начинай с пропусков
  2. Определи часть речи: noun? verb? adjective? adverb?
  3. Смотри на суффикс: -tion = noun, -ly = adverb, -able = adjective и т.д.
  4. Проверь смысл: слово должно логически подходить
  5. Если два варианта похожи — выбирай более формальный / академический (чаще правильно)
Лайфхак для СНГ
Запомни топ-суффиксы: Noun: -tion, -sion, -ment, -ance, -ence, -ity, -ness, -ship Adjective: -able, -ible, -al, -ic, -ous, -ful, -less Verb: -ize, -ise, -en, -fy Adverb: -ly
